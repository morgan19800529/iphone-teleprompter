# 镜头说

面向中文创作者的 AI 直播 / 口播助手。输入一个主题，AI 写出带留人节点的稿子，直接进入提词器开播。

纯前端 PWA，无后端、无账号，脚本只保存在本机。

在线地址：https://morgan19800529.github.io/iphone-teleprompter/

## 已实现

- 口播稿编辑与自动本地保存
- 3 秒倒计时
- 自动滚动、暂停、重播
- 滚动速度与字体大小调节
- 镜像显示
- 阅读引导线
- 横屏适配
- 离线缓存
- 添加到 iPhone 主屏幕后可作为独立 App 使用
- 录制时防止屏幕自动息屏（Wake Lock）
- 轻点提词区暂停 / 继续
- 上下滑动手动调整位置，松手后自动续滚
- 完整 PWA 图标（主屏幕图标不再空白）
- 脚本库：多条口播稿保存、切换、重命名、删除
- 导入 TXT / Markdown，自动清洗 Markdown 语法为纯口播文本
- AI 生成直播 / 口播稿：自动产出标题、提纲、留人节点、互动问题与全文
- 生成结果直接落入脚本库，一键开始提词

## 本地运行

浏览器安全策略要求 PWA 通过 HTTP/HTTPS 运行：

```bash
python3 -m http.server 8080
```

打开 `http://localhost:8080`。

## 部署到 GitHub Pages

1. 将全部文件上传到仓库根目录。
2. GitHub 仓库进入 Settings → Pages。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main` 与 `/root`。
5. 保存后等待 Pages 地址生成。

## AI 功能配置

点右上角齿轮填入 Anthropic API Key，密钥只保存在本机浏览器。

> **注意**：浏览器直连仅适用于自己使用。对外发布时必须改为服务端代理，否则密钥会被他人取走盗刷。
> 代码已预留代理路径：在设置中填入 `endpoint`，即改为请求自己的服务端，上层逻辑无需改动（见 `ai.js` 的 `callModel`）。
